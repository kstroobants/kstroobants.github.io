Research 1

Efficient Machine Learning for mmWave Hand Gesture Recognition

As part of a research master's thesis at Imec, a leading semiconductor research center, I investigated machine learning approaches for hand gesture recognition using mmWave radar data, targeting resource-constrained IoT devices where inference speed, model size, and power consumption are critical. Research question: Can alternative machine learning models achieve competitive accuracy with significantly lower computational and energy costs?

Independent researcher responsible for the entire research project, from problem definition to implementation and evaluation.

  • Built complete ML pipeline starting with MNIST dataset for validation
  • Worked with proprietary mmWave hand gesture datasets from Imec
  • Implemented Multi-Class Gaussian Processes with scalable libraries
  • Evaluated different kernel choices and approximation strategies
  • Used CNNs as baseline for performance comparison
  • Optimized for training time, inference speed, and parameter count
  • Working with high-dimensional, noisy radar data
  • Balancing model accuracy with computational efficiency for edge deployment
  • Limited compute and power budgets for target IoT devices
  • Designing custom features from raw radar signals
  • Working in Linux-based remote compute environments

Demonstrated that Gaussian Process models could achieve competitive performance with significantly fewer parameters than CNN-based approaches. Identified critical trade-offs between model expressiveness and computational efficiency. Showed the viability of non-deep-learning approaches for specific embedded ML use cases, providing insights relevant to energy-efficient edge AI systems. Demonstrated ability to work independently on open-ended technical problems and systems-level thinking that balances accuracy against performance constraints.
